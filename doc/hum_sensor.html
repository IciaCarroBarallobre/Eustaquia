    hum_sensor (eustaquia v0.1.0)

    View Source

    Module for reading soil moisture from a capacitive sensor via I2C.

    This module communicates with an Adafruit soil moisture sensor (or similar) connected to the GRiSP board. It uses the grisp_i2c library to send a read command and retrieve a 16-bit moisture value.

    Example usage:

    1> hum_sensor:read_moisture().
Moisture: 850
{ok, 850}

    read_moisture()

    -spec read_moisture() -> {ok, non_neg_integer()} | {error, term()}.

    Reads soil moisture from the I2C sensor.

    Opens the I2C bus, sends a command to the sensor, and reads a 2-byte value representing the soil moisture level.

    Returns:

    • {ok, Moisture}: The measured humidity value (0-65535)
    • {error, Reason}: If communication fails.
